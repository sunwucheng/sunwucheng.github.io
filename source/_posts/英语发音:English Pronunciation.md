- 元音——气流不受阻碍的音
	- 单元音: 单元音发音过程中口型保持不变
	- 双元音: 两个单元音组成，所有双元音都有明显的口型滑动；双元音在英音中非常夸张且更加圆润
		- 合口双元音：嘴往合的方向走，发音应读成一个(读的过程中嘴从张开向合住并拢的方向发展)
			- 由第一个音向第二个音自然过渡
			- 滑动动作大，且连续
			- 前重，后轻；前长，后短；前清楚，后模糊
		- 集中双元音
			- 由*`[i]`*/*`[e]`*/*`[u]`*向*`[ə]`*滑动
			- *`[eə]`*里第一个音开口略大于*`[e]`*
- 辅音——气流受到阻碍的音
	- 爆破音: 形成阻碍 → 憋气 → 解除阻碍，气流爆破冲出
		- 清辅音: 声带不振动
		- 浊辅音: 声带振动
	- 摩擦音
		- 清辅音: 声带不振动
		- 浊辅音: 声带振动
	- 破擦音
		- 清辅音: 声带不振动
		- 浊辅音: 声带振动
	- 鼻音
	- 舌边音
	- 半元音: 也是一半的辅音，介于元音和辅音之间，功能类似辅音


# 1. 元音音标——单元音

## 英音发音方法

|      |      |      |
| :--: | :--- | :--: |
| *`[i:]`* | 长元音；舌尖轻轻抵下齿龈；略扁唇微笑 | *cheese, beef, please, sweet, tea, sheep, C, G* <br> *beach，sheet，leak，sheep，leave* |
| *`[ɪ]`*  | 短元音；比*`[i:]`*嘴角略收小一点 | *sit, pit, hit, ship, big* <br> *bitch，shit，lick，ship，live* |
|      |      |      |
| *`[e]`*  | 短元音；舌尖轻抵齿龈；上下齿一个食指头宽 | *bed, head, pet, bread, red, send* |
| *`[æ]`*  | 开口大；舌尖抵下齿龈；上下齿两指宽 | *bad, map, fantastic, sad* |
|      |      |      |
| *`[ʌ]`*  | 短元音；在微笑的基础上略微将口开大一点 | *cup, bus, come, trouble, blood, number, love* |
| *`[a:]`* | 长元音；松弛自然，口开到最大，不要噘嘴 | *car, heart, bark, park* |
|      |      |      |
| *`[ə:]`* | 长元音；微笑，扁唇 | *her, work, first, birthday, university* |
| *`[ə]`*  | 短元音；略微开口，喉咙里自然发出轻微的音 | *famous, idea, of, delicious, China* |
|      |      |      |
| *`[ɔ]`*  | 短元音；略微收圆嘴唇，有绷紧的感觉 | *hot, pot, not, got, coffee* |
| *`[ɔ:]`* | 长元音；收圆嘴唇，噘嘴 | *small, law, all, talk, cause*|
|      |      |      |
| *`[u:]`* | 长元音；小圆唇；轻轻噘嘴，不要太用力 | *room, do, blue, movie, true* |
| *`[u]`*  | 短元音；小圆唇；嘴唇稍微翘起 | *good, could, put, cook* |

##  英音发音技巧

He said `/e/` he is very `/æ/` sad because no one is on his side `/aɪ/`.

`[æ]`发音比`[e]`更大，有点呕吐的感觉；`[æ]`在英音和美音中的区别不是很明显，只是美音比英音发音特别往两边咧
`[ʌ]`和中文拼音的`a`很像，区别在于`[ʌ]`是个短音
`[a:]`美音发音没那么靠后；英音发音像唱美声，很靠后 (Papaa，Smart，Afternoon，Demand，Task)

***the a sound*** 字母a的发音出现在-ss, -st, -th, -ff, -of, -nce, -nt等前时，一般美音读为/æ/；英音保持[a:]发音
```
- can't
- fast
- master
- a glass of water
- plant
- answer
- bastard*
```
***The o sound*** 美音发音口型相对随意点——稍微大点，英音发音嘴噘得非常圆、非常用力：英音[ɔ]，美音[a]
```
- I'm hot.
- I've got it.
- Do you like hotpot?
- You did a great job.
- It's not very hot out there.
- Life is like a box of chocolates.
```

## 英音发音方法

|      |      |      |
| :--: | :--- | :--: |
| *`[eɪ]`*  | 第2个音比*`[e]`*小一点、比*`[ɪ]`*大一点 | *make, cake, take, snake, bait, eighth* |
| *`[aɪ]`*  | 第2个音比*`[e]`*小一点、比*`[ɪ]`*大一点 | *Hi, nice, side, bike, bite, eyes, ice, like*  |
| *`[ɔɪ]`*  | 第2个音比*`[e]`*小一点、比*`[ɪ]`*大一点 | *boy, toy, Roy, noise*  |
| *`[əu]`*  | 自然过渡，双唇渐渐收圆，注意结尾不要读成*`[u:]`* | *so, no, rose, don't, home*  |
| *`[au]`*  | 自然过渡，双唇渐渐收圆，注意结尾不要读成*`[u:]`* | *ouch, cow, now, loud, mouth, round*  |
| *`[iə]`*  |                 | *here, ear, dear* |
| *`[eə]`*  |                 | *air, fair, hair* |
| *`[uə]`*  | 现在的*`[uə]`*的英音发音倾向于读*`[ɔ:]`* | *tour, poor, sure* |

## 英音发音技巧

什么时候`[əu]`发音更短一点: (1)在`[əu]`后面有`[l]` (e.g., told, old)时; (2)一个单词中有好几个音节(元音)时

## 英音发音方法

|      |      |      |
| :--: | :--- | :--: |
| *`[p]`* | 形成阻碍时的口型：双唇闭拢 | *play，Potter，pop，cop，develop* |
| *`[b]`* | 形成阻碍时的口型：双唇闭拢 | *boy，Bob，bat，bad* |
|      |      |      |
| *`[k]`* | 形成阻碍时的口型：舌后部抬高紧贴软腭 | *cat，king `/kiŋ/`，Christimas* |
| *`[g]`* | 形成阻碍时的口型：舌后部抬高紧贴软腭 | *got，go，great* |
|      |      |      |
| *`[t]`* | 形成阻碍时的口型：舌尖抵上齿龈 | *ten，top，today* |
| *`[d]`* | 形成阻碍时的口型：舌尖抵上齿龈 | *day，dog，done `/dʌn/`* |

## 英音发音技巧

△爆破音后面不要加“额”！！！
△失去爆破：前面是一个爆破音+后面是一个爆破音（或别的辅音）时，前面这个音几乎听不到，但是口型得做出来

动词加ed什么时候发t/d/id的音: 清后清，浊后浊(包括元音)，t/d后面是/id/
```
asked `/t/`，hoped `/t/`，dressed `/t/`，lived `/d/`，lied `/d/`，waited `/id/`，ended `/id/`
```
清辅音p/t/k/(tr)/什么时候可以浊化(不送气): 一般在s后面，尤其是当s在词首，或者s前有重音符号
```
student，sport，speak，sky，stress，street，excuse /ɪkˈskjuːs/ me
```

# 5. 辅音音标——鼻音+舌边音+半元音

## 英音发音方法

|      |      |      |
| :--: | :--- | :--: |
| *`[m]`* | 双唇闭紧抿嘴，从鼻腔出气，声带振动 | *make，money，team* |
| *`[n]`* | 舌尖抵上齿龈，呈开口状态，空气只能从鼻腔出，声带振动 | *name，line，nine，thin* |
| *`[ŋ]`* | 中度开口，唇形自然，在类/k/、/g/舌位，气流从鼻腔流出 | *long，young，sing，thing* |
|      |      |      |
| *`[l]`* | 清晰音：舌尖抵上齿龈，略微张口，气流从舌的两侧流出 <br> 含糊音：舌尖抵上齿龈，圆唇，呈凹型，气流从舌侧流出 | *late，clear，little，bottle，smile* |
| *`[r]`* | 撅嘴，卷舌，发“若”音 | *right，rich，marry，borrow，bridge* |
|      |      |      |
| *`[j]`* | 开始发音的时候软腭上升，双唇向两旁伸展呈扁平形状 <br> 舌前部向硬腭抬起至原因/i:/或/i/的高度 <br> 声带振动，然后迅速向后面的元音滑动 | *yes，unit，young，beautiful，million* |
| *`[w]`* | 开始发音的时候软腭上升，双唇拢圆而且聚拢得很小 <br> 舌后部向软腭抬起至元音/u:/或/u/的高度 <br> 声带振动，然后迅速向后面的元音滑动 | *way，wish，swim，window，will* |

## 英音发音技巧

`[j]`、`/u/`还是`/ju:/`：字幕u和ew在/d/、/n/、/t/等音之后，美音倾向于读`/u:/`，而英音倾向于读`/ju:/`
```
单词           美音        英音
due           /du:/       /djuː/
produce       /prəˈduːs/  /prəˈdjuːs/
attitude      /ˈætɪtuːd/  /ˈætɪtjuːd/
student       /ˈstuːdnt/  /ˈstjuːdnt/
new           /nuː/       /njuː/
Stupid
opportunity
tune
suit
```

`[l]` darl or clear ？
```
①共性：
l单独出现在元音前，美英音都要卷舌：like, let, play
l出现在元音后面，英音不卷舌，发含糊l：smile, will, feel
②特性：l或ll出现在两个元音中间
UK：只发一个清晰的l
US：先发一个含糊l，再发一个清晰l：smilling, feeling, willing
```

***The r sound***：r在元音前，美英音都要卷舌；r在元音后，英音不卷舌，没有“儿”化音
```
were
word
world/whirled
wore/war
where/wear
air
fire→元音哑巴E，字母本身音（tape, cake …）
```

什么时候卷舌(AmE): 美音里只有单词有r时才儿化，除了colonel /ˈkəːnl; NAmE ˈkəːrnl]/
```
where, idea, famous, New Jersey, quiet, burn, morning, jealous, care, first
```
